## Development

1. Copy `.env.sample`

```shell
> cp .env.sample .env
```

2. Edit `.env`

```shell
TWITTER_CONSUMER_KEY=
TWITTER_CONSUMER_SECRET=
```

3. Run

```shell
# npm
> npm run ex

# yarn
> yarn ex
```
